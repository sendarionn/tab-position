# tab-position

Chrome で新しく開くタブの位置を変更する Manifest V3 拡張機能

コードは Codex を使用して作成した

## Chrome に読み込む

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. この `tab-position` フォルダを選ぶ

## 動作

- ツールバーの「tab-position」から設定画面を開く
- 新しいタブを開く位置を次の4種類から選択
  - 左端
  - 現在のタブの左
  - 現在のタブの右
  - 右端
- 左端を選んだ場合、固定タブのすぐ右に開く
- 設定の変更は自動で反映
- 設定は Chrome Sync に保存
- 同じ Google アカウントでログインしている Chrome 間で設定を同期

## 補足

- 通常の Chrome ウィンドウで作成されたタブが対象
- 新しいタブを作成した直後に指定位置へ移動
- Chrome の仕様により移動前の位置が一瞬表示される場合がある
- 固定タブの並び順は変更しない
