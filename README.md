# Tab Position

Chromeで新しく開くタブの位置を変更できる、Manifest V3対応の拡張機能です。

## インストール

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダ（`tab-position`）を選択する

ツールバーの拡張機能アイコンをクリックすると、次の位置を選択できます。

- 左端（固定タブがある場合は、そのすぐ右）
- 現在のタブの左
- 現在のタブの右
- 右端

設定はChrome Syncに保存され、同じGoogleアカウントのChrome間で同期されます。

## 補足

- 通常のChromeウィンドウで作成されたタブが対象です。
- Chromeの仕様上、新しいタブが作成された直後に移動するため、一瞬だけ元の位置に表示されることがあります。
- 固定タブの並び順は変更しません。
